# 역할: 크리에이티브 디렉터가 정의한 설계도

## 타겟 사용자 (Persona)
- **서원토건 김무빈 차장**: 25개 현장 IT 총괄, 영상 제작 경험 없음
- **Pain Point**: 채널 3개를 동시 운영해야 하는데 영상 1편에 수 시간 소요
- **Goal**: "영상 만들어줘" 한마디로 3채널에 맞는 완성 영상 자동 생성

## 핵심 수치 (숫자의 마법)
- 기존 영상 제작: 평균 **4~6시간/편**
- 자동화 후 목표: **8분 이내 완성**
- 운영 채널: **3개** (서원토건 / 쥬시톡 / 말하지않는것들)
- 파이프라인 단계: **10개 에이전트**
- CapCut JSON: **원클릭** 다운로드
- 비용: 영상 1편당 **약 5,000원** (API 비용)
